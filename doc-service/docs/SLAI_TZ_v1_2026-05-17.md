# [НАЧАЛО ОРИГИНАЛЬНОГО ТЗ]

> ТЗ ниже — продакт-контракт от SLAI, любые изменения требуют их согласия.

## 0. Резюме

parsdocs принимает скан/PDF документа, возвращает структурированный JSON
с распознанными полями. SLAI не занимается OCR — это полностью зона
ответственности parsdocs. SLAI отвечает за matcher (привязку к
Transportation/Transfer) и UI.

**Контракт:**
- in: PDF/JPG/PNG + hint о типе документа
- out: webhook с extracted JSON + confidence

**Что важно для SLAI:**
1. Стабильный JSON-контракт
2. Per-field confidence
3. Нормализация (ИНН, госномер, даты)
4. Honest errors (null вместо выдуманного значения)

## 1. Контракт ответа

```json
{
  "version": "v1",
  "job_id": "uuid",
  "status": "done | needs_review | failed",
  "document_type": "invoice",
  "confidence": 0.94,
  "extracted": { /* зависит от типа */ },
  "_normalized_fields": {
    "seller.inn": "7707083893",
    "buyer.inn": "5024079777",
    "vehicle.plate": "А123АА777"
  },
  "_field_confidence": {
    "seller.inn": 0.99,
    "seller.name": 0.94,
    "items[0].name": 0.71
  },
  "target_entity_hint": "Transportation | Transfer | null",
  "metadata": { "echo": "of POST /jobs metadata" },
  "needs_review": false,
  "raw_text_preview": "first 500 chars of OCR for debugging"
}
```

## 2. Confidence

- `confidence ≥ 0.85` → auto-предложить привязку
- `0.60 ≤ confidence < 0.85` → пометить «требует проверки»
- `confidence < 0.60` → `status='needs_review'`, очередь оператора

## 3. SLA

| Параметр | MVP | Прод |
|---|---|---|
| Время / doc | ≤ 90 сек | ≤ 30 сек |
| Throughput | 5 doc/min | 60 doc/min |
| Размер PDF | до 20 MB | до 50 MB |
| Страниц | до 10 | до 30 |
| Формат | PDF/JPG/PNG | + TIFF/HEIC |
| Availability | 95% | 99.5% |

## 4. 8 open questions для parsdocs

1. **Multi-document PDF** — счёт+УПД+СФ в одном PDF: будете разделять?
2. **Версионирование** — `add field = compat, rename = v2`?
3. **Retry job_id** — endpoint `POST /jobs/:id/retry`?
4. **OCR-only режим** — для нестандартных форм через `hint: "raw_ocr"`?
5. **Шрифты и языки** — кириллица + латиница ок, китайский нужен для AliExpress?
6. **Rate limit per token** — какой порог?
7. **Длительные обработки** — polling `GET /jobs/:id` или ждать webhook?
8. **Storage retention** — оригинал PDF удаляется при `redact_pii=true`?

## Acceptance критерии (резюме)

### Invoice
- `seller.inn`, `buyer.inn`, `document_number`, `total_with_vat` → **≥ 95%**
- items[*], bank-реквизиты → ≥ 80%

### Transport request
- `client.inn`, `carrier.inn`, route addresses → **≥ 90%**
- `vehicle.plate` нормализован → ≥ 95%
- даты в ISO → ≥ 85%

### TTN
- shipper/consignee/carrier ИНН, `vehicle.plate` → **≥ 95%**
- `total_weight_kg` ±1 кг → ≥ 95%
- driver.fio, route → ≥ 90%
